# Flipkart Sales Dashboard

An interactive Excel dashboard built to analyze Flipkart sales performance. The dashboard provides insights into sales trends, product categories, payment methods, quantity sold, and customer ratings using Pivot Tables, Pivot Charts, and Slicers.

## Key Insights
- Total Orders: 700
- Total Products: 27
- Total Categories: 7
- Total Quantity Sold: 2162
- Total Sales: ₹53,378,505
- Average Rating: 3.05

## Dashboard Components
- Sales and Prices Distribution
- Payment Method Analysis
- Category-wise Sales
- Year-wise Quantity Sold
- Product Rating Analysis
- Interactive Filtering using Slicers
